# 加载提示

cp 在 Vue 的原型链上注册了加载提示的方法，调用后将显示全屏遮罩防止用户点击，提示用户等待程序处理，默认显示内容为 '正在加载'，可传入自定义提示语作为参数。

组件采用单例模式，可重复调用进行显示内容的设置，demo 如下：

// index.js
new App({
  mounted() {
    this.$wait()
    // this.$wait('提示')
    // 关闭提示
    // this.$wait.over()
  }
})

WARNING

cp 中存在名为 cp-wait 的加载提示组件，一般情况下不建议对其直接操作，开发中需要注意不要引用该组件，或注册同名组件。
