# 对全局环境的影响

# 命名空间

cp 中大部分的私有变量，如 class 名、组件名等，都带有 cp- 的前缀，一般情况下不会与其他框架或类库冲突，另外也有部分半私有的属性名带有下划线作前缀，如使用了 select 对象后将在 window 对象上注册 __cp_select_group 数组。除较为常用的变量外都进行了私有化命名处理，但在开发中还是需要注意一下变量冲突的问题。

# API 兼容

为了方便开发及兼容运行环境，cp 对几个常用的 API 做了较为简单的兼容，但采取的兼容方案是很简陋的，如需更加完善的兼容，请参考 mdn 上的建议方案，以下为兼容列表：

  • Array.map
  • Array.filter
  • Array.includes
  • String.includes

仅在运行环境不支持以上 API 时，才对其进行兼容，优先使用原生的方法。

# DOM 事件

  • touchmove

微信浏览器及 iOS 环境下阻止了 touchmove 的默认事件使其无法触发滑动，需要触发时请添加 stop 修饰符以阻止事件冒泡，如需创建竖直滚动内容区请使用 roll 对象。

  • contextmenu

默认禁用了 cp 实例挂载节点的长按事件，但不影响 img 标签长按的相关事件，需要触发长按菜单时请添加 stop 修饰符以阻止事件冒泡。

# img 标签

cp 是在移动端中诞生的，移动端开发中单屏应用占了很大的份额，而单屏项目很多时候都是由一大堆图片堆砌而成，为了方便开发，在 cp 中对全部的 img 标签做了处理：

  • wrap 对象及不可滚动的 page 对象内的 img 标签默认为绝对定位
  • roll 对象及可滚动的 page 对象内的 img 标签默认为静态定位

添加 freedom 属性即可解除定位限制，所以如果在开发过程中发现 img 标签的定位有异常，可以查看是否是由 freedom 引起的，在控制台中查看元素的的时候可以很容易发现这个问题的存在，但在开发时注意这个问题，可以省去一点麻烦。demo 如下：

<img freedom>

微信某些内核版本的浏览器中，点击 img 标签会启用图片浏览模式，长按会启用额外操作菜单，在很多情况下这并不是我们想要的。在 cp 中通过 css 禁用了 img 的触碰，添加 touchable 属性可以解除这一限制，此时可以对其进行点击或长按等操作，所以如果在 cp 中要为元素绑定点击事件，需要这样子写：

<!-- 添加 prevent 防止启用图片浏览模式 -->
<img touchable @click.prevent=Todo>

WARNING

需要注意的是，在富文本内容中可能包含需要长按识别的二维码图片，而富文本内容是很麻烦而且也没必要涉足的部分，只需通过 css 选择器，为该区域内的 img 添加规则即可。如保证富文本内不含需要长按菜单的图片，则可以跳过设置：

.rule img
  pointer-events auto

# Vue 自定义指令

  • suit

用于元素适配的指令，详情请查看元素自适应模块。

  • roll

用于滚动对象相关事件及样式的绑定，不建议直接使用，如需使用滚动内容请参考 roll 对象。

# Vue 过渡动画

参考 Vue 文档中的 transition 组件部分，动画表现为淡入淡出：

<transition name=cp-fade></transition>

# Vue 原型链方法

  • $wait

用于显示加载提示，详情请参考加载提示模块。

# 工具方法

开发中常见的逻辑处理，为了方便使用，对其进行的封装，开发中需要注意变量冲突的可能性：

// document.getElementById 的代理
$id('id')

// 传入数组和长度，返回按长度分割的数组集合
SliceArray([1, 2, 3], 2) // [[1,2], [3]]

// 传入数字，大于 9 的将返回原值，小于或等于 9 的将在前补零返回
FormatNum(9) // '09'
FormatNum(10) // 10

// 传入可被 Date 对象识别的初始值以及输出格式，返回格式化的时间
// 依赖 FormatNum 方法
FormatTime('2018/12/25', '例子') // '例子'
FormatTime('2018/2/25', 'y年M月d日') // '2018年02月25日'
FormatTime(1545708597863, 'M月d日 h:m:s') // '12月25日 11:29:57'

// 传入内容及显示时长，弹出吐司式提示，非单例模式，反复调用将出现多个提示
Toast('Content', 10000) // 自定义显示时长,默认为 1500 ms
Toast(['Content', 'Test']) // 支持传入内容数组,将分行居中显示内容

# 工具样式

常用的样式封装：

.tl0
  top 0
  left 0

.wh100
  width 100%
  height 100%

.p_center
  left 50%
  transform translate(-50%)

.p_vetically
  top 50%
  transform translateY(-50%)

.p_middle
  top 50%
  left 50%
  transform translate(-50%, -50%)

.f_center
  display flex
  justify-content center

.f_vetically
  display flex
  align-items center

.f_middle
  display flex
  align-items center
  justify-content center

.h_wrap
  display -webkit-box
  -webkit-box-orient vertical
  overflow hidden

.h_nowrap
  overflow hidden
  white-space nowrap
  text-overflow ellipsis

.bg
  @extend .tl0
  @extend .wh100
  z-index -1

@keyframes cp-keyframe-rotate
  form
    transform rotate(0)
  to
    transform rotate(360deg)
