# 更多命令

# 网页元数据

aml 提供 meta 命令用于配置页面的元数据，输出结果将插入到模板的 meta 插槽中：

<!--
  meta name content               // 配置插入的 meta 标签，多次配置将追加引入
  meta name2 'content2 content3'
-->

输出为：

<!DOCTYPE HTML>
<meta charset=utf-8>
<title></title>
<meta name=viewport content='width=device-width, initial-scale=1, user-scalable=0'>
<meta name=name content=content>
<meta name=name2 content='content2 content3'>
<script>
  document.getElementsByTagName('html')[0].style.fontSize = innerWidth / 7.2 + 'px'
</script>

# 网页图标

aml 提供 icon 命令用于配置页面的图标：

<!--
  icon test.ico  // 默认值为空字符串，单行命令，重复配置将覆盖，可不配置 icon
-->

输出为：

<!DOCTYPE HTML>
<meta charset=utf-8>
<title></title>
<meta name=viewport content='width=device-width, initial-scale=1, user-scalable=0'>
<script>
  document.getElementsByTagName('html')[0].style.fontSize = innerWidth / 7.2 + 'px'
</script>
<link rel='shortcut icon' type='image/x-icon' href=test.ico>

# 百度统计

aml 提供 hmt 命令用于配置页面的百度统计信息：

<!--
  hmt 1234567890  // 传入百度统计的 id，默认值为空字符串，单行命令，重复配置将覆盖，可不配置 hmt
-->

输出为：

<!DOCTYPE HTML>
<meta charset=utf-8>
<title></title>
<meta name=viewport content='width=device-width, initial-scale=1, user-scalable=0'>
<script>
  _hmt = []
  document.getElementsByTagName('html')[0].style.fontSize = innerWidth / 7.2 + 'px'
</script>

<script src=//hm.baidu.com/hm.js?1234567890></script>

# 页面调试

aml 推荐使用 eruda 来调试页面，且提供 eruda 命令用于添加调试工具：

<!--
  eruda // 存在时将为页面添加调试工具，单行命令，可不配置 eruda
-->

输出为：

<!DOCTYPE HTML>
<meta charset=utf-8>
<title></title>
<meta name=viewport content='width=device-width, initial-scale=1, user-scalable=0'>
<script>
  document.getElementsByTagName('html')[0].style.fontSize = innerWidth / 7.2 + 'px'
</script>

<script src=//cdn.jsdelivr.net/npm/eruda></script>
<script>eruda.init()</script>
