# 闲谈

AIOS Markup Language（下称 aml）的雏形是 2018.05.25 那天，为了从千篇一律的 HTML 中得到解脱，基于 node.js 开发的 HTML 预处理器，由于是自定语法，所以起了这个名，意思是我的标记语言。

最初那版的 js 源码只有 93 行，内容也仅包括几个标签的转义和图片路径的补全，还有一些资源插入，其中自然是有些 bug 的，页面模板也是写死，但已经可以在项目中省下成百上千的代码量了。

同年的六月十七，重构了一次源码，还增加了读取模板文件的功能，之后在九月三号那天还增加了 aml 转 aspx 和 cshtml 的功能，百度统计和文件编码处理功能都被压缩为一行命令，自此真正达成了不写 HTML 的成就。配合 Clever Page 的话，开发效率之高，让人闲得无聊。

虽然 aml 在那时已经成熟，也算是立了个项，但代码上因为能力问题，不能做得很优雅，所以版本号一直是 1.0.0，只作为私人项目在用。

截至写下这段话，aml 的 js 源码已经达到了 233 行，因为定位的原因，文档一直没有写，直到写完了 Clever Page 的文档，才想着，还是写点什么吧，怎么说也是自己生的，于是就有了这堆内容。

时间节点记得那么清，主要还是因为，发了朋友圈。
